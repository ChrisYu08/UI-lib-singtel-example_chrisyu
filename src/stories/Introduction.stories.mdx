# Singtel UI component Library

Singtel UI component Library helps developer to develop UI components in both web and mobile.


## How to change component size for different devices.
1. web component
  the default size for viewport is web PC viewport

2. tablet
  change the viewport to Tablet viewport and then click the remount component button

3. mobile
  change the viewport to mobile viewport and then click the remount component button
